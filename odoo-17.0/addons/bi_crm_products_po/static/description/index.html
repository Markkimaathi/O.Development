Create RFQ - Request for Quotation from Lead and Pipeline Odoo Apps

Create RFQ/PO from Lead/Pipeline Odoo app simplifies the process of transforming a lead or pipeline opportunity into a request for quotation (RFQ) or purchase order (PO). With just a few clicks, this app allows users to add products to a lead and eventually convert it into RFQs or POs, complete with scheduled delivery dates. Additionally, it provides a seamless integration with leads, easy access to created purchase orders directly from a smart button within the lead, and the lead will be linked with the purchase order.

Features

Create RFQ from Lead/Pipeline

User can add products to lead and create RFQ from lead.

Create Purchase Order from Lead/Pipeline

User can add the product to the lead and create a purchase order from lead.
Demo Request Video Tutorial

Create RFQ from Lead/Pipeline

User can add products on the lead form under the "Products" tab and create a request for quotation by clicking the "PURCHASE ORDER" button.

On the wizard, user needs to select "Purchase State as Request For Quotation" and enter the schedule date and click the "CREATE" button.

On the lead/pipeline form view user can see and access created request for quotation from the smart button.

User can see created RFQ from lead with the same products group by the vendor and lead also link with created RFQ.

Create Purchase Order from Lead/Pipeline

User can add products on the lead form under the "Products" tab and create purchase order by clicking the "PURCHASE ORDER" button.

On wizard, user needs to select "Purchase State as Purchase Order" and enter the schedule date and click the "CREATE" button.

On the lead/pipeline form view user can see and access created purchase orders from the smart button.

User can see created purchase order from lead with the same products group by vendor and the lead also link with created purchase order.
