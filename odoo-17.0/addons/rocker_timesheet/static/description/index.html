Rocker Timesheet

hr_timesheet module supercharged. Drop tasks/work done from Project/Task tree directly to calendar view.
Time_off compliant

Works Odoo 17, 16, 15, 14 & Odoo 13 (Odoo 12 does not have searchpanel)

- Hour Bank Report Start date (reporting min Start date)
- Hour Bank Report Initial Saldo at Start Reporting Start date
- Start date and Initial saldo currently only in version 17

- Version for Odoo 16
- Fixed some multi Company problems (Hour Bank Report)
- Removed Calendar right pane filter (problem with odoo 16)
- Removed user defaults (calendar start & stop, slot & def mode (Odoo16 problem)

- Import Public Holidays to Odoo Calendar. Supports many countries. Data imported from Enrico 2.0
